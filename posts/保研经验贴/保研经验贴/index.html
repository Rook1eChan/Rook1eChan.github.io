个人背景#

**学校：**北京普通一本

**GPA（rk）：**排名2

**英语：**CET-4：611；CET-6：583

**所获荣誉：**校三好、优团（国奖用处大，其它荣誉没啥用）

0.写在前面#

**你是否应该选择保研？**除了保研，升学的道路还有考研和留学。我们在努力之前，应该先问清自己的目标是什么。

在时间上，保研意味着三年的持续付出，你需要在课内卷绩点，在课外卷竞赛和论文，比其它同学付出更多。而考研则可以享受前三年的大学生活，只紧张最后一年。当然，保研er可以更早解脱。当考研er还在紧张备考时，你已经上岸了。

在去向上，考研的上限更高，只要你分数够高，就能去到对应的好的院校。保研的下限更高，一般大家都能去比本校高一个档次的院校。但是保研的上限较低，保研不是单一用分数评价人，同届有大量手握竞赛论文的大神和你竞争，而且有的院校根本不收双非四非的学生，所以想保研去tp华五这样的顶级院校还是挺难的。

在风险上，保研的风险较小，三年的时间你有很多机会去提升自己，在夏令营预推免的时候你可以联系大量的老师，机会很多。而考研就是看分数，如果发挥不好也没有别的办法了。这也是我选择保研的原因，经历过高考以后，我再也不想参加这种一局定胜负的考试了。

**今年的保研形势？**今年和往年有很大变化：

  1. 大部分夏令营停办或无效力，使得剩下开夏令营的学校人数暴涨，oq现象更严重，仍然是20%的人拿80%的offer。相当于少了一个占坑的机会。而且今年是第一年改革，很多人由于变化带来的担忧选择投递档次更低的院校，导致后面的人没了位置。
  2. 由于夏令营大家没有offer，导致预推免的门槛也水涨船高。大家的offer比往年档次要低。
  3. 院校保研率增加，但接收名额并未增加，保本校的学生会占掉更多名额。
  4. 推免系统开放提前。而且今年是统一发复试、预录取通知，学生统一确定（往年是各院校按自己的时间），之间间隔时间变长为两天。这一点利于鸽子起飞，利于候补的学生拿offer。

个人预测，夏令营没效力、保研名额增加会成为趋势。

2.2竞赛#

竞赛用处不大，老师一般不会问，但有总比没有好。

数学建模类：美赛、国赛

创新创业类：大创、挑战杯等

科研实践类：各种人工智能、计算机竞赛


2.3论文#

对于双非学生，有论文意味着清楚科研的完整流程，能提高个人竞争力。个人感受，有论文的佬最后去向都能高一个层次。

当然论文质量要保证，投CCF B或CCF C，EI就算了，数模产出的论文千万别写。

论文一二作才有用，二作往后一般认为贡献极小。


2.4机试（编程题）#

2.4.1类型#

机试一般分为两种题型：

  1. 写完整程序 ：需要在IDE（dev c++、pycharm、vscode等）上面写一个程序，有输入输出，运行后从cmd读取输入，然后把输出打印出来。c++程序示例如下：

    #include<bits/stdc++.h> # 导入包
int main(){
    int a,b;
    cin>>a>>b;  # 读取输入
    cout<<a+b;  # 打印输出
    return 0;
}

    蓝桥杯就是这种形式。深大、软件所、计算所、天大也是这种形式。

  2. 补全函数：以力扣上面的题目为代表，函数已经定义好了，你只要写其中的处理逻辑，将答案return就行了。苏大是这种形式，有的老师在考察时会让你做力扣，也会涉及到。


2.4.2学习路径#

个人建议有时间的话可以以赛代练。编程比赛有蓝桥杯、百度之星等等。还可以考一个CCF CSP认证，有的学校可以用来抵机试分数，比如北航。以赛代练比较有动力，适合刚入门时找找感觉。直接在蓝桥杯网站上练题就行了，牛客网也可以，当时软件所在这上面考的。

另外一个必刷的就是LeetCode 热题 HOT 100，基本上涵盖了编程题所有基础题型，考察频率很高。个人建议，按照灵神的基础算法精讲来刷。

刚开始刷的时候必然挫败感很强，因为题型都没见过，根本没思路。可以设一个20分钟的倒计时，如果倒计时结束还没做完就看参考答案，然后做好笔记。当你再次遇到相同类型的题目时，就能很快得出思路了，每个类型的题都有其套路的。

2.5项目#

把课程设计包装一下；大学生学术支持计划；竞赛中的项目

要体现出完整的流程，包括研究现状调查、数据收集清洗、系统代码编写、评估、实验现象分析这几个关键步骤。要清楚自己在其中做出的贡献是什么，面试会问到。


2.6英语#

  1. 四六级一定要考，一般都是要过425。而且如果你六级分数很高（570+甚至600+），老师印象会好，确实是这样。
  2. 英语自我介绍背熟练，一般1~2分钟。老师不会仔细听你说的什么，一般你说的时候都在看简历，但是你至少得说流畅了。有的院校会让你英语简要介绍做过的项目，有的会英语问你几个问题（你怎么看待科研的失败、用英语介绍网络安全的一个名词等等）。
  3. 北科还有英语阅读理解，和四六级差不多。

2.7 408#

一般来说老师都会问项目或论文，简历没东西才会问408。所以不用太系统的复习，找一下总结的帖子把高频考点背一下即可。

3.院校分析#

计算机保研天梯榜：

此外还有一些著名的lab，如上海人工智能实验室，中关村人工智能研究院等。

还有今年兴起的三所人工智能院校：上海创智学院、中关村学院、深圳河套学院。一般是和985院校联培的，需要拿到两边的offer合成大offer。


选择院校会考虑以下几个方面：

  1. 地域：南方，还是北方？回家乡，还是去大城市？
  2. 该学校的学科排名如何？可以参考第四、第五次学科评估，CSRankings。
  3. 专业的学费如何，家庭能否负担得起？是否提供宿舍？有的专硕不提供宿舍，只能租房。
  4. 校区所在位置如何？一般靠近公司比较好，方便实习。
  5. 最重要的，导师人品如何？接下来会详述。

4.导师分析#

先明确自己想要什么样的老师，是羊导，还是科研抓的比较紧的导师？然后根据以下几个方面评价一个导师适不适合自己。

  1. 人品如何：人品有问题就是把学生当奴隶使唤，又干活又不发钱，天天PUA，甚至卡毕业。这个需要自己搜，网上有导师评价网和一些表格，知乎贴吧也有一些评价。最真实的就是去联系他的学生，可以通过实验室网站、论文找学生的邮箱，或者小红书、QQ群里问。还需要注意下有没有抢学生一作的现象。
  2. 研究方向：找你之前的方向相匹配的老师。想换方向也可以，需要先学点对应的基础知识。方向可以看个人介绍，或者到Aminer/谷歌学术看近期发的论文来确定。
  3. 导师能力：每年的发文数量、质量；被引次数；title（优青、杰青等）。
  4. 毕业要求：毕业要不要发论文，需要发什么级别的论文。
  5. 实习要求：如果你打算硕士就业，对论文其实没那么看重，几段大厂实习更有用。所以需要问一下老师什么时候可以放实习。
  6. 日常要求：有没有按时打卡、每周组会。
  7. 往届学生就业去向。

在后面套磁、夏令营、预推免选导师时，都按照感兴趣的院校——其中计算机相关的系——专业匹配的导师——各项指标符合自己要求的导师 这个思路来选导师。


5.保研流程#

大一-大三下：保持绩点、参加竞赛、发表论文

1-2月：准备文书材料，开始刷算法题。

3-4月：这个时候可以联系弱com院校的老师。这些院校老师招生话语权大，一般都需要你去做几个月的项目进行考察。比如说北邮，6月份联系其实都有点晚了，老师名额都满了。大概5月往后到8月都可以持续套磁你意向的院校。

6-7月：参加夏令营

8-9月：参加预推免。最后填系统结束。


5.1套磁#

套磁就是联系老师，表达你研究生想跟他进行研究的愿望，让老师留一个招生名额给你。4月份开始就行了，一直到最后结束前都可以持续套磁。

我采取的是海投策略，把目标院校所有专业相关的导师的信息收集起来并打分，按照分数从高到低给老师发邮件，如果有高分老师回复就不用继续套后面的老师了。之所以海投，是因为回复率真的很低，10封邮件大概只有一两个老师回复。所以建议还是多联系，这样才有得选。

5.2夏令营#

信工所：bar不高点击入。两天线上宣讲，一共十一个研究室室，1密码学，4-6人工智能，7-8系统安全，9-10网安，11科教融合。此外每个室都有出名的老师，比较抢手。

可以报两个室的志愿，某个室报的人太多就会筛，比如6、4（人工智能+知乎小红书风评好，太火了，不乏92来的）。我报了6、9，最后6果不其然被筛了，进了9。

4、6、10有额外机试，其余只有面试。9室是5分钟英语+15分钟综合。英语是1-2min自我介绍+两个问题，简介一个你专业的名词、你怎么定义科研的成功。综合就是5min自我介绍，要ppt。只拷打了项目，问我有哪些改进，有无产出。然后问了点专业相关的（敏感端口怎么处理）。幸好没问408，忘完了。

当晚打电话问接不接受直博。后来给了优营，可以去线下参观。

信工所新园区很整洁美观。研究所和高校不同，整体是办公氛围，除了学生还有很多职工。信工所整体上更注重项目，主要是依托重大项目，科研产出偏少。信工所在网安领域的认可度高，如果一心想从事网安行业，来信工所读个硕士是不错的，因为这里毕业较容易，而且硕士找网安工作没有特别看重论文，个人实战能力强也可以。另外信工所补助较高，硕士2000+，博士5000+。一句话，想去网安，且想留北京的，非常推荐去信工所。


东南palm：官网报名，一共有三轮面试，报名哪一轮都行。简历过了会通知你面试。进面的bar很迷，也不是说bg强就一定能进。进面是以第一志愿导师为单位考核，腾讯会议自我介绍，据说只有直博会拷打时间长一些，硕士自我介绍完基本就结束了，三天后等结果。寄。


5.3预推免#

北交：联系到一位不错的NLP老师，结果简历直接被卡，连笔试都没进。

苏大：苏大专硕学硕同样培养，一般是专硕较多，因为毕业要求较低，不需要发论文，只要两个研究点加一个系统，搞完就可以实习去了，老师也喜欢招专硕。学硕需要CCF，三个研究点，难毕业。

苏大流程是先过学院考试，然后老师推荐，就ok了。进组以后就跟这个老师了，然后自己选感兴趣的方向，也可以换方向。

后面有苏大线下机试。过了以后可以联系老师写推荐表。建议是去机试前就联系好，不然后面空名额很少。苏大有超发offer的毛病，有推荐表也可能被鸽，和老师要问清楚。


深大：机试10个题。面试10分钟，英语自我介绍，英语介绍自己的一个项目。详细问了竞赛（美赛，蓝桥，大创，论文）。然后问了有没有具体想研究的方向，夏令营进了没有，该方向有没有调查过相关的老师。面试后联系到一位老师。


计算所：填两个意向研究室。线下机试，6个题，要求是尽量不使用自带的函数，写好注释，按照思路给分。面试主要问了项目，有一些针对项目的扩展，然后408高频考点可以过一下，可能会问到。

5.4正式推免#

9.18开始填个人信息

9.22填三个平行志愿

9.23确认复试

9.25确认拟录取

有铁offer或者候补到的同学到这里就结束了。没有offer的同学可以关注院校的二次推免信息，因为有的院校会被鸽穿，这时候可以捡漏。（点名华东软、中山、中科大十系。信工所、软件所今年也开了二次推免招生）


6.信息源#

小红书：小红书的推荐机制很强，用来搜导师院校评价、搜文书模板、闲暇的时候搜集一些边角料信息很有用。注意不要被上面的焦虑情绪感染了。

保研通知网：更新及时，内容很全。每天记着查看哪些夏令营、预推免更新了。

AMiner：用来搜导师的论文用。和Google Scholar有点出入，主要是不用翻墙比较方便。
